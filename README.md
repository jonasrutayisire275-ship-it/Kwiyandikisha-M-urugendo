# Form Submission System

A complete form submission system with GitHub integration for storing form data and file uploads.

## Features

✅ **User-friendly form** with validation
✅ **File upload support** (up to 5MB)
✅ **GitHub integration** - stores all submissions as GitHub issues
✅ **Admin dashboard** - view and manage all submissions
✅ **Search functionality** - filter submissions
✅ **Export to CSV** - download all submissions
✅ **Real-time stats** - track submissions, gender distribution, file uploads
